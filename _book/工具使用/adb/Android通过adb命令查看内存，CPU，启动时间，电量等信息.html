3、查看应用启动时间

方法1

命令：

adb logcat -c && adb logcat -f /mnt/sdcard/up.txt -s tag

选项说明

-c   清屏
-f   指定运行结果输出文件，默认输出到标准设备（一般是显示器
-s   设置默认的过滤级别为Silent
tag  仅显示priority/tag

更多信息烦请参考 adb logcat -help

例：

先启动app，然后执行如下命令

例： 先启动app，然后执行如下命令 C:\Users\laiyu>adb logcat -c && adb logcat -f /mnt/sdcard/up.txt -s ActivityMana ger

备注：I/ActivityManager： I 代表优先级，ActivityManager代表tag
